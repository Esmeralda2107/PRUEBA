# 🗂️ Competencia en Base de Datos

Este directorio contiene la **entidad de competencia** en su versión estructurada para base de datos, resultado de la integración territorial de la fuente gastronómica ya depurada y clasificada.

En esta etapa, los registros de establecimientos fueron vinculados al sistema territorial del proyecto y organizados de forma compatible con el modelo relacional, permitiendo su posterior agregación y explotación analítica.

La tabla resultante incorpora información relacionada con **identificación del establecimiento, localización, tipología gastronómica, clasificación de competencia e identificadores territoriales**, constituyendo la dimensión de competencia dentro de la base de datos del proyecto.
