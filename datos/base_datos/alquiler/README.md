# 🗂️ Alquiler en Base de Datos

Este directorio contiene la **entidad de alquiler comercial** en su versión estructurada para base de datos, resultado de la integración de la fuente de elaboración propia dentro del modelo de datos del proyecto.

En esta etapa, la información fue organizada para mantener una correspondencia clara con las zonas del área de estudio, permitiendo su incorporación al sistema relacional y su posterior uso en la construcción del dataset maestro.

La tabla resultante incorpora información relacionada con **zona, identificadores territoriales de referencia y precio estimado de alquiler comercial**, constituyendo la versión estructurada de esta dimensión dentro de la base de datos del proyecto.
